# Autonomous Driving in BeamNG

## Goal

This project aims to build an autonomous driving software stack from scratch and validate it using BeamNG simulation.

## Roadmap

### Phase 1
- Vehicle Control
- Steering
- Throttle
- Brake

### Phase 2
- Camera Sensor
- Lane Detection
- Obstacle Detection

### Phase 3
- Path Planning
- Autonomous Navigation

### Phase 4
- ROS2 Integration
- Performance Evaluation

## Technologies

- Python
- C++
- OpenCV
- BeamNGpy
- Git
- ROS2 (Future)
